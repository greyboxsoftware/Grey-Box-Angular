      Software would be almost useless without variables. If you are a programmer then you will use variables in everyday life. In PowerShell variables are really really easy and when I say that it means that PowerShell doesnt require you to specify type of variable i.e. Int, String, Double, etc...

All variables in PowerShell start with the "$" character.

$variableHere = "I am a variable and a string"
$variableNumber = 4

Variable do not have to be declared as a specific type in PowerShell.

      For every beloved DOS command there is a PowerShell command that does it better and in this case the ping command is replaced by Test-Connection. This is very simple and you can do alot but a basic example is as follows.

Test-Connection -ComputerName www.google.com -Count 1
